# eeg-motor-imagery-decoder

## Overview
This project explores EEG-based motor imagery classification using the PhysioNet EEG Motor Movement/Imagery Dataset.

## Objectives
- Process EEG recordings
- Extract motor cortex features
- Train a machine learning classifier
- Build a simple closed-loop BCI demonstration

## Methods
- MNE-Python
- Power Spectral Density 
- Linear Discriminant Analysis 
- Cross-validation

## Results
Current classification accuracy varies depending on train/test split and cross-validation setup.

## Future Work
- Common Spatial Patterns (CSP)
- More advanced classifiers
- Real-time EEG streaming
- Improved closed-loop feedback
